Detta är ett backend-baserat webshop-projekt byggt med C# och Entity Framework. Projektet visar grundläggande e-handelsfunktioner och inkluderar även adminfunktioner för att hantera produkter, kunder och beställningar.

Funktioner
Kundsidan:

*Visa produkter med namn, beskrivning och pris

*Lägg till och ta bort produkter i kundvagnen

*Simulerad checkout

Admin:

*Hantera produkter (lägg till, uppdatera, ta bort)

*Hantera kunder och beställningar

*Full kontroll över webshopens innehåll

Teknologier

*C# (.NET)

*Entity Framework & Dapper

*SQL-databaser på Azure
